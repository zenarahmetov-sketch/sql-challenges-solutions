# SQL Challenges & Solutions

### About
This repository contains my solutions to various SQL challenges from platforms like LeetCode, HackerRank, and DataLemur. The goal is to demonstrate my ability to write efficient queries, handle complex joins, and use advanced SQL functions.

### Tech Stack
* **Dialect:** PostgreSQL / MySQL / SQLite
* **Concepts:** Subqueries, Window Functions (RANK, DENSE_RANK), CTEs, Joins, Aggregations.

### Structure
Each `.sql` file follows a specific format to be compatible with my [SQL Collector](https://github.com/zenarahmetov-sketch/sql-portfolio-collector):
* **Metadata Header:** Title, Difficulty, and Category are defined in comments.
* **Code:** Clean, executable SQL query.

### 🤖 Automation
Data from this repository is automatically parsed and validated by a Python automation tool to be displayed on my personal portfolio website.
